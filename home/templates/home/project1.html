Final Year Project: Next-CS

A Comprehensive Career Guidance Platform for Computer Science Students and Fresh Graduates

🔍 Project Overview

Next-CS is a one-stop career platform tailored for CS students and graduates. It bridges the gap between learning and employment by offering career resources, self-assessment tools, peer interaction, and real-time job listings — all within a unified platform.

🚀 Key Features

📊 Personalized Dashboard

Dynamic dashboard with tailored job insights, career suggestions, and skill progress tracking.

🧠 Quiz Hub

Subject-wise quizzes in HTML, CSS, Python, DBMS, and more to evaluate knowledge and learning gaps.

📚 Learning Hub

Structured roadmaps, curated books, YouTube tutorials, and online courses — all organized by domain.

📝 Resume Builder

Create, preview, and download professional resumes using intuitive form-based templates.

🤝 Connect Campus

Engage with peers via request system and private 1-to-1 chat. Builds community and peer support.

💼 Job Search Integration

Aggregates real-time job listings from public APIs like GitHub Jobs and RemoteOK, with filters and links.

🛠️ Technologies Used

  • Frontend: HTML5, CSS3, JavaScript, Bootstrap
  • Backend: Django Framework (Python)
  • Database: MySQL
  • API Integration: RemoteOK API, GitHub Jobs API
  • Version Control: Git & GitHub
