Project: Craft Studio

A Full-Stack E-commerce Platform for a Stationery Shop

🔍 Project Overview

Craft Studio is a fully functional e-commerce web application built for a stationery shop. It includes category-based product listings, a secure shopping cart, user account system, and an intuitive checkout process — delivering a complete online shopping experience using Django and MySQL.

🚀 Key Features

🔐 User Authentication

Signup, login, and logout functionalities with secure session management and access control.

📦 Dynamic Product Listing

Category-wise item filtering (e.g., Pens, Notebooks, Art Supplies) for an organized shopping experience.

🛒 Shopping Cart System

Users can add/remove items, adjust quantity, and view live totals in a clean cart interface.

📤 Order Placement

Secure order confirmation with Cash on Delivery/UPI options and a success feedback page.

📋 Order History

View detailed list of past orders by date, item summary, and order status.

📱 Mobile Responsive UI

Modern, responsive layout built using Bootstrap 5 and custom CSS for smooth mobile experience.

🛠️ Technologies Used

  • Backend: Django (Python)
  • Database: MySQL
  • Frontend: HTML5, CSS3, Bootstrap 5, JavaScript
  • Templating Engine: Jinja2 (Django Templates)
  • Version Control: Git, GitHub
